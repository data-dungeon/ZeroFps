MADEXPORT:
Exporterar från maya till XXX. För att skapa en MAD måste man sen ochså skapa en mdf och använda madmake.
Alla filer som skapas är text men dom får olika filändels beroende på vad dom innehåller.

Options:
	Export SD (XS):
		Exporterar skellet data till en XXX fil.

	Export AD (XA):
		Exporterar AD till en XXX fil.

	Export MD (XM):
		Exporterar MD till en XXX fil.

	Export MD (Animated) (XMA):
		Exporterar animerad MD till en XXX fil. Alla vertex frames skrivs ned men inga direkta trianglar.



	XXX File Format
Är text fil format som maya exporterar till. En XXX fil består av Skellet Data (SD),Animations
Data (AD), Mesh Data (MD). En modell (MAD) består av 1 SD, x antal MD och x antal AD.

	SD: Är den grundläggande strukturen för skelletet som bygger upp modellen och anger namn
		för alla bones. En modell kan endast ha ett skellet.
	AD:	Är det som får saker att animeras. Är en lista med alla dom ställningar som skelletet ska
		genomgå för animationen. En MAD kan ha hur många animationer som helst.
	MD:	Är dom delar som ritas ut. Varje mesh anger på vilka delar av skelletet som den är kopplad 
		och hur mycket den påverkas av skelletet rörelser. En MAD kan ha hur många olika MD som helst.
		En MD innehåller oftast bara en enda uppsättning vertex då den normalt animeras vi bones. MD 
		stödjer dock även vertex animation med en uppsättning vertex på frame. För att detta ska fungera
		bör alla vertex i MD vara kopplade till endast ett och samma ben.

Nodes

	start nodes

	...
	end

	time x
	<ID> <Pos> <Rot> 
	...
	end

Triangel:

	"texture name"
	[3] <parent ID> <Pos> <Normal> <uv> 
	...
	end

	

	MDF Fil
Denna fil sänds till madmake och den används för att skapa MAD modeller. Den är i text
format den med. Den består av en olika kommandon.

	filetype <SD | AD | MD>
		Typ av fil som ska skapas. 
	
	modelname <pathname>	
		Path och namn på fil som ska skapas. 

	cd <path>
		Set path för soure files.

	cdtextures <path>
		Set path för textures.

	set-sd <path>
		Anger den XXX fil som innehåller skelleton data som ska användas. Kan endast
		förekomma en gång per MAD.

	add-ad <name> <xxx-pathname> <fps> <LOOP>
		name:	Namnet på animation.
		xxx-p:	Namnet på den xxx fil som innehåller animationen.
		fps:	Playback speed på animation.
		LOOP:	Om det är en looping animation.

		Skapar en animation som denna modells ska stödja.

	add-md <meshname> <pathname>
		Skapar en mesh och kopplar den till skelletet. Pathname är sökvägen till XM fil som ska 
		användas.

	add-meshanim <meshname> <animname> <pathname>
		Lägger en vertex animation till en mesh.
		
	include <pathname>
		Laddar och inkluderar en annan mdf fil i denna fil.		
 
	translate
	scale
	rotate

	MADMAKE:
Tar informationen i ett MM-script och skapar modeller. Förljande argument finns.

	Inget		Utan argument så skrivs version och en kort hjälptext ut.

	*.mm		Gör det som skrivits i MM filen.

	mm name		Med denna parameter så skapas en standard mm med namnet name.mm i samma katalog där MADMAKE körs.
				Alla filer i nuvarande katalogen som stödjs länkas in i denna mm fil.
	
	*.3ds *.mad	Exportera från 3ds till mad.
	*.mdl *.mad	Exportera från mdl till mad.
	*.md2 *.mad	Exportera från md2 till mad.
	
				
MAD:
För att skapa en modell i spelet så skapar man ett MAD object först och anger ett skellet att använda. Sen

Skellet och animationer laddas alltid men mesh object laddas bara om dom används.

add-md f f.md

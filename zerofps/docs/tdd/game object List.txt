Game Object i ZeroFps:
 SkyBox:	Ritar en skybox. 
 HojdMap:	Är en hojdmap för landskaps terräng. 
 WorldMap:	Är spelvärlden man spelar i och den innehåller flertalet olika hojdmaps. Dessa laddas
			dynamiskt beroende var spelaren / spelarna är. På servern laddas endast den data som krävs för
			server code (collision, fysik). 
  
 SFX:		Billboards (pint, line). 
 Sound:		Avger ett ljud.




*** Propertys ***
 
 Position:			Anger objektets position och riktning.

 Sound/
  Sound:			Spelar upp ljud.
 
 Render/
  Light:			Objekt har en egen ljuskälla som lyser upp omkring den.
  RenderType:		Hur objekt ritas ut. Normals, None, Editor.
  Self Ambient:		Min light level för objekt.
  Transparancy:		Markerar ett object som genomskinligt. Alla genomskinliga object ritas ut sist.
  Mad:				En modell med skellet animationer.
 
 SFX/
  ParticleEmitter:	Skapar partiklar.
  ParticleForce:	Kraftfält som påverkar partiklar.
 
 Attach:			Kopplar ett object till ett annat.

Fysik:
 Typ				Hur objectets position,rotation och rörelser updateras.
	NONE:			Rör sig aldig.
	GHOST:			Rör sig men påverkar inget annat.
	WALK:	
	FLY, TOSS, BOUNCE:

Collision:
 Type:				Avgör modell som används för att avgöra collision på detta object.
	POINT, SPHERE, LINE, AABBOX, OOBBOX:

 Dimensions:		Beskriver formen på collision object.

 Effect:			Vad som händer när en kollision sker.
	NONE:
	TRIGGER:
	BLOCK:

 Awerness:			Hur medveten en AI är om något annat object.

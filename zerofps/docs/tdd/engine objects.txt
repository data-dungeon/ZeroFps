Alla större system (Render, Sound, Console,...) ska placeras i ZeroFps namnsystem. Det
används för att man ska kunna komma åt object från olika platser i programmet utan
att behöva använda globala variabler. Varje object får ett eget namn som man kan använda.

Detta namn är oftast samma sak som klass namnet för object som det endast finns ett av.
Detta system används även för att sända messages (kommandon) till object.

How to - ZeroFps

1: Ladda ned alla program och usla libs man behöver.
 A: STLport			4.5.3	http://www.stlport.org/
 B: Glut			3.7	http://www.opengl.org/developers/documentation/glut.html
 C: Simple DirectMedia Layer 		http://www.libsdl.org
	C1:	SDL		1.2.5	http://www.libsdl.org
	C2: SDL_image		1.2.2	http://www.libsdl.org/projects/SDL_image/
	C3: SDL_mixer		1.2.4	http://www.libsdl.org/projects/SDL_mixer/
	C4: SDL_net		1.2.4	http://www.libsdl.org/projects/SDL_net/
 D: OpenALSDK			1.0	http://www.openal.org/
 E: SourceForge Development Tools:
	Windows:
		CVS:		http://www.wincvs.org/
		SSH:		http://sfsetup.sourceforge.net/#sfsetup
	Linux:
 F: Lua	4.0.1
	Source code		http://www.lua.org/ftp/lua.tar.gz	
	Precompiled binaries
		Windows:	http://www.tecgraf.puc-rio.br/lua/ftp/binaries/Windows/	
 G: toLua 4.0			ftp://ftp.tecgraf.puc-rio.br/pub/users/celes/tolua/tolua-4.0.tar.gz

 Z: Ogg vorbis skräp : 		http://www.vorbis.com/download.psp
	OBS! länka mot dessa libs: ogg_static.lib 
				   vorbis_static.lib 
				   vorbisfile_static.lib	

2: Installera allt man laddat ned.

3: Sätt upp sökvägar för alla usla libs.


4: Be till diverse gudar och offra diverse offer.
	Håll tummarna.
	Knäpp händerna.
	Slå huvet i skärmen
	Knäck tangentbordet och bit av mussladden
	Om det fortfarande inte fungerar upprepa föregående steg tills godkänt resultat uppnås.

Install:
 copy/past content of usertype.dat into /common/msdev98/bin/usertype.dat
 copy/past content of autoexp.dat into /common/msdev98/bin/autoexp.dat 

What:
 autoexp.dat:
	Change the way some of the zerofps data types display in the 
	debuger.

 usertype.dat:
	Turns some of the basic data types in zerofps to display as
	user defined keywords.
